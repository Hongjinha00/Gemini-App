# 만들어야 할꺼
Electron으로 제작된 Gemini 데스크톱 앱

# 필요한 이유
매번 웹브라우저에서 키기 귀찮기 때문에 GPT 앱처럼 Gemini도 웹 앱과 기본 레이아웃은 동일한 대신 단축키로 호출하고 화면 위에 고정할 수 있어야됨 

# 필요한 기능

- 웹사이트와 동일한 Gemini 웹페이지
- 단축키로 프로그램을 열고 닫을 수 있어야됨
  - 앱이 열릴 시 기본 세팅은 화면 중앙에서 약간 위 세로 589 가로 442 창크기 
  - 앱이 닫혀있을때는 최소한의 성능만 사용해서 백그라운드에 켜놓기
  - 윈도우 시작시 같이 켜져야됨
  - 앱을 단축키로 불렀을때 속도가 빨라야됨
  - 앱을 단축키로 열때 창의 크기, 위치, 작업 내역은 모두 닫기 전과 동일해야됨
    - 대신 컴퓨터 재시작시 기본 셋으로 초기화
- 창이 모든 창 맨 위에 고정돼어야됨
- 윈도우 시작시 실행, 단축키, 창 고정은 설정으로 바꿀 수 있어야됨
- [Gemini App](https://gemini.google.com/app)과 [google ai studio](https://aistudio.google.com/)사이를 전환 할 수 있어야됨 
  - 이건 컴퓨터가 재시작되도 마지막에 선택한게 나와야됨 
- 창 제어 버튼 있는 상단 바에 설정, ai스튜디오 전환, 뒤로/앞으로 가기 버튼이 있어야됨
